# OSI C1P PCjs Emulator Reference Manual
**Version:** 0.1  
**Status:** In Development  
**Maintainer:** rgjohn & ChatGPT  

This manual documents the OSI Challenger 1P (Superboard II) emulator within the PCjs project and all custom extensions added by the author.

- Custom additions, including new tools and UI extensions
